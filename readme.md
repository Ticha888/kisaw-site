Я добавил полноценную загрузку файлов, доступную при нажатии на иконку скрепки ("Paperclip"). Это реализовано на обеих страницах: `index.html` и `brand-package.html`.

Возможности и ограничения, которые я заложил:
1.  **Типы файлов**: Настроено ограничение на выбор текстовых файлов (`.txt`, `.pdf`, `.doc`, `.docx`) и изображений (любых типов). Выбор других типов заблокирован на уровне окна выбора (с помощью атрибута `accept`).
2.  **Множественный выбор**: Вы можете выбирать сразу несколько файлов (и за несколько заходов, новые файлы добавляются к существующим, дубликаты игнорируются).
3.  **Ограничение по размеру**: Прописан лимит на максимальный размер одного файла в **20 МБ**. Если файл превышает его, пользователю выведется стандартное предупреждение (Alert).
4.  **UI - Интерфейс отображения**: После добавления файлы автоматически появляются в виде списка над текстовым полем и кнопками (внутри специально отведенного блока, который становится видимым только в статусе прикрепленных файлов).
5.  **Динамическое обновление**: Текст-подсказка ("helper-status") автоматически обновляется, показывая количество прикрепленных файлов и их суммарный размер.
6.  **Удаление**: Возле каждого загруженного файла есть кнопка крестика "✕" — она полноценно работает и позволяет удалять ошибочно выбранные файлы, обновляя весь остальной UI (размер и счетчики).

Можете проверить, нажав на иконку скрепки — откроется диалоговое окно добавления файлов!
